# Security Policy

## Reporting a Vulnerability

If you discover a security issue in this plugin, please report it privately to Synta instead of opening a public issue.

- Email: `info@synta.io`
- Website: `https://synta.io/`

Please include:

- A short description of the issue
- Steps to reproduce it
- Any affected files or configuration
- The potential impact

We will review the report and respond as quickly as possible.
